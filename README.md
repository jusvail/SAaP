# SAaP

[WinUI3项目]

股票每日溢价跟踪&分析程序

需要Win10/Win11版本10.0.17763.0以上

从Windows Store安装:

[![从Windows Store安装](/Assets/msstore.svg)](https://www.microsoft.com/store/apps/9P9PVZS13C97)

## 依赖项

- python运行环境
- TDX（需每天更新日线数据）
- .NET 6.0运行环境
